title: Password Reset
---

Manage password reset functionality for user accounts.

## Forgot Password

<api>POST /authentication/forgot-password</api>

Initiate the password reset process by sending a reset token to the user's email.

### Authentication

This endpoint does not require authentication.

### Request Body

<ParamField body="email" type="string" required>
  User's email address. A password reset token will be sent to this email if it exists in the system.
</ParamField>

### Response

<ResponseField name="isSuccess" type="boolean">
  Indicates whether the password reset email was sent successfully.
</ResponseField>

<ResponseField name="succeeded" type="boolean">
  Alias for isSuccess. Indicates whether the operation succeeded.
</ResponseField>

<ResponseField name="error" type="object">
  Error information if the operation failed.
  
  <Expandable title="error properties">
    <ResponseField name="code" type="string">
      Error code identifier.
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Human-readable error description.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="errors" type="object[]">
  Array of error objects if the operation failed.
</ResponseField>

### Error Responses

<ResponseExample>
```json 400 Bad Request - Validation Error
{
  "isSuccess": false,
  "succeeded": false,
  "error": {
    "code": "Validation.Failed",
    "description": "El email no es válido"
  },
  "errors": [
    {
      "code": "Validation.Failed",
      "description": "El email no es válido"
    }
  ]
}
```
</ResponseExample>

<RequestExample>
```bash cURL - Forgot Password
curl -X POST https://api.sapfiai.com/authentication/forgot-password \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com"
  }'
```

```csharp C# Client - Forgot Password
using var httpClient = new HttpClient();
var request = new ForgotPasswordCommand
{
    Email = "user@example.com"
};

var response = await httpClient.PostAsJsonAsync(
    "https://api.sapfiai.com/authentication/forgot-password",
    request
);

var result = await response.Content.ReadFromJsonAsync<Result>();

if (result.Succeeded)
{
    Console.WriteLine("Password reset email sent successfully");
}
else
{
    Console.WriteLine($"Failed to send reset email: {result.Error.Description}");
}
```
</RequestExample>

<ResponseExample>
```json 200 Success - Forgot Password
{
  "isSuccess": true,
  "succeeded": true,
  "error": {
    "code": "",
    "description": ""
  },
  "errors": []
}
```
</ResponseExample>

## Reset Password

<api>POST /authentication/reset-password</api>

Reset the user's password using the token received via email.

### Authentication

This endpoint does not require authentication. Use the reset token from the email.

### Request Body

<ParamField body="email" type="string" required>
  User's email address.
</ParamField>

<ParamField body="token" type="string" required>
  Password reset token received via email.
</ParamField>

<ParamField body="newPassword" type="string" required>
  New password for the account. Must meet the following requirements:
  - Minimum 8 characters
  - At least one uppercase letter
  - At least one lowercase letter
  - At least one number
  - At least one special character
</ParamField>

<ParamField body="confirmPassword" type="string" required>
  Password confirmation. Must match the newPassword field.
</ParamField>

### Response

<ResponseField name="isSuccess" type="boolean">
  Indicates whether the password reset was successful.
</ResponseField>

<ResponseField name="succeeded" type="boolean">
  Alias for isSuccess. Indicates whether the operation succeeded.
</ResponseField>

<ResponseField name="error" type="object">
  Error information if the operation failed.
  
  <Expandable title="error properties">
    <ResponseField name="code" type="string">
      Error code identifier.
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Human-readable error description.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="errors" type="object[]">
  Array of error objects if the operation failed.
</ResponseField>

### Error Responses

<ResponseExample>
```json 400 Bad Request - Invalid Token
{
  "isSuccess": false,
  "succeeded": false,
  "error": {
    "code": "PasswordReset.InvalidToken",
    "description": "Invalid or expired reset token"
  },
  "errors": [
    {
      "code": "PasswordReset.InvalidToken",
      "description": "Invalid or expired reset token"
    }
  ]
}
```

```json 400 Bad Request - Validation Error
{
  "isSuccess": false,
  "succeeded": false,
  "error": {
    "code": "Validation.Failed",
    "description": "La contraseña debe tener al menos 8 caracteres"
  },
  "errors": [
    {
      "code": "Validation.Failed",
      "description": "La contraseña debe tener al menos 8 caracteres"
    }
  ]
}
```
</ResponseExample>

<RequestExample>
```bash cURL - Reset Password
curl -X POST https://api.sapfiai.com/authentication/reset-password \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "token": "reset-token-from-email",
    "newPassword": "NewSecureP@ssw0rd",
    "confirmPassword": "NewSecureP@ssw0rd"
  }'
```

```csharp C# Client - Reset Password
using var httpClient = new HttpClient();
var request = new ResetPasswordCommand
{
    Email = "user@example.com",
    Token = "reset-token-from-email",
    NewPassword = "NewSecureP@ssw0rd",
    ConfirmPassword = "NewSecureP@ssw0rd"
};

var response = await httpClient.PostAsJsonAsync(
    "https://api.sapfiai.com/authentication/reset-password",
    request
);

var result = await response.Content.ReadFromJsonAsync<Result>();

if (result.Succeeded)
{
    Console.WriteLine("Password reset successful");
}
else
{
    Console.WriteLine($"Password reset failed: {result.Error.Description}");
}
```
</RequestExample>

<ResponseExample>
```json 200 Success - Reset Password
{
  "isSuccess": true,
  "succeeded": true,
  "error": {
    "code": "",
    "description": ""
  },
  "errors": []
}
```
</ResponseExample>
